## 版本状态: beta

计划中的大部分功能特性已经到位，但仍可能存在bug。在到RC阶段前，API仍可能改变，建议不要在生产环境中使用。

## 快速开始

``` sh
npm install -g @vue/cli
# 或者
yarn global add @vue/cli

vue create my-project
```

有关常见任务的详细指南和每种插件的详细用法，请参阅[完整的文档](./zh-cn/README.md)。
